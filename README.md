# 🛒 RedStore – Responsive E-Commerce Web Application

A fully responsive front-end e-commerce website built using **HTML, CSS, and JavaScript**. The project provides a modern online shopping experience with product browsing, product details, shopping cart UI, and user authentication pages.

---

## 🚀 Features

* ✅ Responsive design for mobile, tablet, and desktop
* ✅ Attractive homepage with featured and latest products
* ✅ Product listing page
* ✅ Product details page with image gallery
* ✅ Shopping cart interface
* ✅ Login and Registration forms with animated transitions
* ✅ Mobile navigation menu toggle
* ✅ Product ratings using Font Awesome icons
* ✅ Clean and user-friendly UI

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Font Awesome**
* **Google Fonts (Poppins)**
